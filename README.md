# Workflow Library

## Overview

The **Workflow Library** is the companion workflow collection for **Siril & Seti Astro Suite Pro Processing Companion**.

This repository contains developed workflow files, documentation, and reference material for deep-sky astrophotography using **Seti Astro Suite Pro** and the **Siril Workflow Assistant**.

The workflows reflect the best practices of others and my 60+ years of practical astrophotography experience combined with modern AI-assisted image processing software techniques.

## Processing Philosophy

The workflow library is built around several core principles:

* Preserve astronomical signal before enhancing aesthetics.
* Use AI tools thoughtfully and conservatively.
* Favor scientifically accurate color whenever practical.
* Validate every major processing step.
* Maintain repeatable, documented workflows.
